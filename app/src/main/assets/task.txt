Задание(для Андроид) Нужно разработать приложение, которое должно делать следующее:

- есть JSON файл, расположенный по ссылке(http://ibecsystems.com/test.json).
Она содержит список стран и столицу страны вместе с несколькими крупнейшими городами.
Каждый город содержит в себе количество жителей. Нужно взять данные оттуда и сохранить в простую базу данных.

- экран должен содержать следующее:
    - список стран в выпадающим списке. При нажатии на страну должен выпадать список городов этой страны,
    расположенных по алфавиту, причем столица страны должна выделяться отдельным цветом. Надо создать два отдельных списка.
    Одна из них будет значит город, из которого мигрируют(город А), а второй город, в который мигрируют(город В)
    - Должен быть TextField. В нее надо записать количество мигрантов. Также должна быть кнопка «Провести миграцию».
     По ее нажатию надо проверить, есть ли в городе А столько жителей,сколько указано в текстфилде.
      И если есть, то этих жителей надо перевести в город В. После этого надо переписать базу данных

-       лэйауты должны отличаться при смене ориентации экрана и должны быть адаптивными для разных размеров экранов.
        Будет плюсом если это будет реализовано через Fragment

-       выбранные города и содержимое текст филда должны сохраняться и обратно подставляться при смене ориентации экрана;

-       все миграции должны записываться в базу данных для того, чтобы потом можно было посмотреть в истории;


-       отдельная страница для просмотра истории миграции.

- каждую минуту количество жителей в каждом городе должно расти на 5%, а в столице на 10%.
    После каждого изменения кол-во жителей должно записываться в базу данных

Желательно, но не обязательно:

- Создать Service, который будет уведомлять когда количество жителей одного города превысит 5, 10, 20, 50, 80, 100 миллионов
